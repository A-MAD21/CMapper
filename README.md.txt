# CMapper

A small web/backend project (Python + HTML/CSS/JS) that includes:
- `Backend.py` (backend entry)
- `Templates/` (HTML templates)
- `Static/` (CSS/JS/assets)
- `Modules/` (python modules/helpers)
- `generated_maps/` (output directory)

## Quick start

### 1) Create a virtual environment
```bash
python -m venv .venv

.venv\Scripts\activate
# macOS/Linux:
